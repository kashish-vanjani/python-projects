#  Python Calculator (CLI)

A simple command-line calculator built in Python.  
This program performs basic arithmetic operations like addition, subtraction, multiplication, and division.

---

##  Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/) with zero division handling
- Runs until user chooses to exit
- Input validation for numbers

---

##  How to Run

1. Make sure Python is installed.  
2. Run the script in terminal:

```bash
python calculator.py

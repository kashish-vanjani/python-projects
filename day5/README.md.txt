#  Python File Organizer

A simple command-line tool in Python that automatically sorts files in a folder into subfolders based on their file extensions.

---

##  Features
- Organizes all files in a given directory by their extension  
- Creates folders for each extension (e.g., `pdf`, `jpg`, `py`)  
- Handles name collisions by renaming duplicates (e.g., `file(1).txt`)  
- Works on the current folder or any specified path  

---

##  How to Run

1. Make sure Python is installed.  
2. Run the script in terminal:

```bash
python file_organizer.py

#  Python Dice Roller Simulator

A simple command-line dice roller built in Python.  
This program simulates rolling one or more dice with any number of sides.

---

##  Features
- Roll any number of dice
- Choose number of sides per die
- Shows individual rolls and total sum
- Simple and lightweight (runs in terminal)

---

##  How to Run

1. Make sure Python is installed.  
2. Run the script in terminal:

```bash
python dice_roller.py
